Munchkin for Android

Developer - Android

Munchkin for Android was built for Computer Science 309 taught by Professor Mitra. Munchkin, the original, is a partial turn based multiplayer card based game. The objective is to get to level 10 before everyone else by hurting them or helping yourself. The project consisted of Android clients, and a Java server that would communicate between the clients to update the UI respectively. Overall the project was successful and won a class award for one of the top 5 best projects.
